# F004 - 選手一覧ページ

**ステータス:** ✅ 実装済み
**優先度:** P1

## 概要
全選手をカード形式で一覧表示するページ (`/players`)。

## 機能要件

### 表示内容
- PlayerCardsコンポーネントを再利用（ダッシュボードと同じ）
- 全選手のカードをグリッドレイアウトで表示
- 各カード: 背番号, 名前, PPG, RPG, APG
- トップ選手にバッジ表示

### インタラクション
- カードホバーでスケール・輝き効果
- カードクリックで `/player/[number]` に遷移

## データソース
- `getPlayerSummaries()` でサーバー側取得

## 受け入れ条件
- [x] 全選手が表示される
- [x] カードクリックで選手詳細に遷移
- [x] レスポンシブ対応
