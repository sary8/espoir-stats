# F007 - デザインシステム

**ステータス:** 実装済み
**優先度:** P1

## 概要
Tailwind CSS v4 ベースのデザインシステム。ダークテーマ + 紫アクセント。

## カラーパレット

### ベースカラー
- 背景: `#0a0a0f`
- 前景: `#e5e5e5`
- カード背景: `rgba(255, 255, 255, 0.05)`
- カードボーダー: `rgba(255, 255, 255, 0.1)`

- 紫ライト: `oklch(0.714 0.203 293.57)`

- pink `#F472B6`, red `#F87171`, cyan `#22D3EE`
- yellow `#FBBF24`, indigo `#818CF8`, teal `#2DD4BF`
